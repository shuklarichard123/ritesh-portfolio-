Slide 02
Executive Summary
🎯

Problem Statement

External users need a formal, auditable process to request IP whitelisting access to internal applications. Manual email-based processes lack traceability and approval tracking.

✅

Solution

A fully serverless web application hosted on AWS. Users submit structured requests via a web form. All requests are stored in DynamoDB with a unique Request ID for tracking.

💡

Key Benefits

Zero server management · Near-zero cost (~$1/month) · Auto-scaling · Full audit trail · Accessible globally via CloudFront CDN · CI/CD automated deployment.

In scope: Request submission, storage and retrieval  ·  Out of scope: Automated firewall rule provisioning (handled by network team post-approval)

Frontend Layer
CloudFront serves HTML/CSS/JS from S3 over HTTPS with global edge caching. SPA routing handled client-side.
API Layer
API Gateway exposes /records endpoints. Lambda proxy integration routes POST, GET requests. CORS enabled for browser calls.
Data Layer
DynamoDB stores each request with a UUID. GSI on username enables filtered queries. Pay-per-request billing — no idle cost.

Slide 05
Request Data Flow
📤 Submit Flow
1

User opens the form

Browser loads index.html from CloudFront edge. Assets served from S3 via CDN.

2

User fills & submits

Enters name, email, IP/CIDR, port, environment, protocol, justification and approver.

3

Browser calls API Gateway

JavaScript sends POST /records with JSON payload to API Gateway HTTPS endpoint.

4

Lambda processes request

Python handler validates input, generates UUID request_id, records timestamp.

5

Stored in DynamoDB

Record written to user-tracker table. Request ID returned to browser and shown to user.

📥 Retrieval Flow
1

Admin opens View Requests tab

Clicks "Load Requests" — optionally filters by requester name.

2

GET /records called

With username param: DynamoDB Query on GSI. Without: full Scan. Sorted newest first.

3

Table rendered

Shows Request ID, Name, Organisation, IP, Environment, Duration, Status, Date.

4

Lookup by ID

GET /records/{request_id} fetches the full record with all fields including justification and approver.

🔒 Security Controls

🔐

HTTPS Everywhere

CloudFront enforces HTTPS. All browser-to-API traffic encrypted in transit.

🛡️

IAM Least Privilege

Lambda role limited to PutItem, GetItem, Scan, Query on user-tracker table only.

🪣

Private S3 Bucket

Bucket is not public. Content served exclusively through CloudFront OAC.

📋

Audit Trail

Every request stored with timestamp, requester details and approver in DynamoDB.

📊

CloudWatch Logging

All Lambda invocations logged to /ecs/portfolio for monitoring and debugging.
